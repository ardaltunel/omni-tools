# Arda Tools

Arda Tools, Arda Altunel projelerini tek bir statik web panelinde birlestiren HTML, CSS ve JavaScript uygulamasidir.

## Uygulamalar

- Advanced XOX
- Chess
- Chess Coach
- Crypto Price
- GitHub User
- Gradient Generator
- Numble
- Pass Generator
- QR Code Generator
- Text To Speech
- Translate
- Type Test
- Weather App
- Wordle

## Kullanim

Projeyi basit bir yerel sunucu ile calistirabilirsiniz:

```bash
npx serve .
```

Araclar sidebar uzerinden secilir. Liste, yeni arac eklendiginde JavaScript tarafinda otomatik alfabetik siralanir.
